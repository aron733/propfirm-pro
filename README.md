# PropfirmPro - Plateforme de Trading Prop Firm

🚀 Une plateforme moderne de trading propriétaire avec simulation de trades, dashboard interactif et gestion des challenges.

## 🎯 Fonctionnalités

✅ **Système d'authentification** - Inscription/Connexion avec localStorage
✅ **Dashboard de trading** - Interface complète pour passer des trades
✅ **Simulation de trades** - Exécution de trades avec profit/perte simulés
✅ **Gestion des comptes** - Suivi du solde, profit et historique
✅ **Landing page attrayante** - Présentation des challenges et tarifs
✅ **Design responsive** - Adapté à tous les appareils
✅ **Design moderne** - Glassmorphism, gradients, animations

## 📋 Pages disponibles

- **index.html** - Page d'accueil
- **login.html** - Connexion/Inscription
- **dashboard.html** - Panel de trading
- **pricing.html** - Tarifs et challenges

## 🚀 Démarrage rapide

1. Cloner le repository
2. Ouvrir `index.html` dans un navigateur
3. Créer un compte sur la page d'inscription
4. Accéder au dashboard pour commencer à trader

## 🛠️ Technologies

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (Vanilla)
- LocalStorage pour la persistence des données

## 📱 Responsive Design

Optimisé pour tous les appareils (mobile, tablet, desktop)

## 📝 Notes

- Les données sont stockées en localStorage (données locales au navigateur)
- Les trades sont simulés avec des profits/pertes aléatoires
- Pas de backend - application full frontend
